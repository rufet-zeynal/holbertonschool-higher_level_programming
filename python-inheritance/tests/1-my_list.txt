====================================
1-my_list module
====================================

Import the module:
>>> MyList = __import__('1-my_list').MyList

Create an instance:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)

Check the original list:
>>> print(my_list)
[1, 4, 2, 3, 5]

Check the sorted print:
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Check that original list is unchanged:
>>> print(my_list)
[1, 4, 2, 3, 5]
